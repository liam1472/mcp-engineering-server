---
description: Index API routes for web projects
allowed-tools: MCP
---

Run the MCP tool `eng_routes` to index all API routes in the codebase.

Supports:
- Express / Fastify (Node.js)
- NestJS decorators
- Flask / FastAPI (Python)
- Gin / Echo / Chi (Go)
- ASP.NET Core (C#)

Usage: /eng-routes

The routes are saved to `.engineering/index/routes.yaml` for reference.
